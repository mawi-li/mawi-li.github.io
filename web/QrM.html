QrM

A web library to convert a file (mp4/mp3/png) into QR codes and recover it by scanning.

ConvertRecover

Developer Usage

You can use the library via direct link or by downloading the js file.

Via Link
<script src="https://mawi.li/assets/js/qrm_core.js"></script>
Download File
Download qrm_core.js
API
QrM.encodeFileToPayloads(file)
QrM.payloadsToPngDataUrls(payloads, { width }, onProgress)
QrM.estimateForFile(file)
const session = new QrM.RecoverSession()
session.addPayloadText(text)
session.assembleBlob()

Note: The library uses local QR dependencies to avoid tracking/prevention issues.
